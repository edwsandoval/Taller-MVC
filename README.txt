El proyecto realiza registros en la Base de Datos "db.sqlite3"

Pasos para ver funcionalmente el proyecto (windows): 

1)Iniciar el entorno virtual init:
	a)Desde consola dirigirse a la direccion Entornos/init/Scripts/
	b)Escribir "activate" y presionar enter
	c)En caso de no funcionar verificar la instalacion de la libreria virtual env
	  (para instalarla ejecutar la siguiente linea: pip install virtualenv).

2)Correr el servidor local
	a)Desde la consola ir al directorio raiz
	b)Dirigirse a la direccion Proyecto/django_crud_mvc/
	c)Correr la siguiente linea: python manage.py runserver

3)Entrar desde el navegador a http://localhost:8000/
